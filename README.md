# 💳 Payment Gateway UI

A clean, modern payment gateway user interface inspired by popular UPI applications like **PhonePe**, **Paytm**, and **Google Pay**. This project demonstrates frontend development fundamentals with an interactive, beginner-friendly payment flow.

---

## ✨ Features

- 🎨 **Modern, Clean UI** - Professional design with smooth animations
- 💰 **Interactive Payment Flow** - Multi-step payment experience
- ✅ **Payment Success Screen** - Confirmation and transaction details
- 📱 **Responsive Design** - Works seamlessly on all devices
- 🚀 **Beginner Friendly** - Pure HTML, CSS, and JavaScript (no frameworks)
- ⚡ **Zero Dependencies** - Lightweight and fast
- 🎭 **UX Focused** - Intuitive user interface inspired by industry leaders

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/vipulyadav0024-commits/Payment-Gateway-project-1.git
   cd Payment-Gateway-project-1
   ```

2. **Open the project**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server (recommended)
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

3. **That's it!** No build process required.

## 💻 Usage

### Basic Flow

1. **Launch** - Open `index.html` in your browser
2. **Enter Amount** - Input the payment amount
3. **Add Recipient** - Fill in recipient details
4. **Verify OTP** - Enter the verification code (demo only)
5. **Confirm** - Review and complete the transaction
6. **Success** - View payment confirmation screen

### Interactive Elements

- ✏️ **Form Inputs** - Input fields for amounts and details
- 🔘 **Buttons** - Action buttons with hover effects
- ⌨️ **Keyboard Support** - Full keyboard navigation
- 📱 **Mobile Friendly** - Responsive touch-friendly interface

## 🎨 Customization

### Change Colors
Edit `style.css`:
```css
:root {
  --primary-color: #6C63FF;      /* Change primary color */
  --success-color: #51CF66;      /* Change success color */
  --error-color: #FF6B6B;        /* Change error color */
}
```

### Modify Payment Flow
Edit `script.js` to:
- Add/remove steps
- Change validation rules
- Update success messages

### Adjust Styling
- Modify fonts in `style.css`
- Update button styles
- Change animations and transitions
